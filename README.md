# 0xFEclicker

Larping 20CPS in the big '26!
## Features

* **Toggle keybind** — enable or disable the autoclicker with a configurable key.
* **Hold-to-click** — clicking is automated only while the left mouse button is held.
* **Configurable CPS** — choose the target clicks per second.
* **Legacy combat support** — ignores the modern vanilla attack cooldown so it can operate on servers using older 1.7.10/1.8.9-era Spigot combat behaviour.


## Usage

1. `RSHIFT` or `.0xFE or /0xFE` in game to open the GUI.
