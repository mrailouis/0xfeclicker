## What is 0xFEclicker?

**0xFEclicker** is a lightweight Fabric mod that automates left clicks while you hold the attack button. It is designed to feel simple and unobtrusive, with an in-game ClickGUI for controlling its behaviour.

---

## Features

<details>
<summary><b>Configurable toggle key</b></summary>
<br>
Choose the key that enables or disables the autoclicker whenever you need it.
</details>

<details>
<summary><b>Hold-to-click</b></summary>
<br>
Clicks are automated only while the left mouse button is held and the game has captured your mouse.
</details>

<details>
<summary><b>Configurable CPS</b></summary>
<br>
Set the target click rate from the ClickGUI, from disabled at 0 CPS up to 25 CPS.
</details>

<details>
<summary><b>Legacy combat support</b></summary>
<br>
Ignores the modern vanilla attack delay so the autoclicker can operate on servers using older 1.7.10/1.8.9-era Spigot combat behaviour.
</details>

---

## Usage

1. Press <kbd>Right Shift</kbd>, or run `.0xfe` or `/0xfe` in chat, to open the ClickGUI.
2. Choose a toggle key and set your preferred CPS.
3. Enable **AutoClicker**, then hold the left mouse button while playing.

Your settings and GUI position are saved automatically.

## Requirements

- Minecraft **1.21.11**
- Fabric Loader **0.19.3** or newer
- Fabric API
- Fabric Language Kotlin
- Java **21** or newer
